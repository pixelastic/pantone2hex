# pantone2hex

Converts Pantone color names to hex.

## Usage

```sh
$ ./pantone2hex 104C
#af9800
```

## Dependencies

`pantone2hex` used [jq](http://stedolan.github.io/jq/) as a dependency to parse
the JSON embedded in the script. You can `git clone` it from the previous link.
